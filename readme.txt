== Description ==

How it works:

1. Click 'Tools > Pagination Test Drive'.

2. Set it up.

3. Enjoy!

== Installation ==

1. Upload 'pagination-test-drive' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
